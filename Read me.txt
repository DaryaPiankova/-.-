Инструкция по установке:
Установите необходимые инструменты
1. Node.js и npm
2. Docker
3. MongoDB и PostgreSQL
4. RabbitMQ

Запуск:
Теперь, когда у вас есть все необходимые инструменты, выполните следующие шаги для запуска фронтенда:

Перейдите в папку frontend:
cd frontend

Установите зависимости:
npm install

Запустите приложение:
npm start
